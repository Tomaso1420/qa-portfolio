# Sudoku Project — REST API

Repository: https://github.com/Tomaso1420/sudoku-project

## Описание проекта

Sudoku Project — backend-приложение для генерации и решения Sudoku через REST API.
Проект поддерживает асинхронную обработку задач через очередь.

## Что реализовано

* REST API для создания задач;
* генерация Sudoku;
* решение Sudoku;
* получение задачи по id;
* хранение статусов задач;
* обработка статусов pending / processing / done / failed;
* асинхронная обработка через worker;
* взаимодействие с PostgreSQL;
* запуск сервисов через Docker.

## Моя роль

Разработчик и тестировщик учебного backend-проекта.

## Что я проверял

* создание задачи на генерацию Sudoku;
* создание задачи на решение Sudoku;
* получение результата по id задачи;
* изменение статусов задачи;
* обработку успешных сценариев;
* обработку некорректных входных данных;
* работу API вместе с worker и очередью задач.

## Стек

Python, FastAPI, PostgreSQL, RabbitMQ, Docker, REST API.
