# DocValidator Bot — Legal Document Assistant

Repository: https://github.com/Tomaso1420/doc-validator-bot

## Описание проекта

DocValidator Bot — проект ИИ-бота для анализа юридических документов, выявления рисков и помощи в подготовке юридических текстов.

## Что реализовано

* загрузка пользовательских документов;
* обработка документов разных форматов;
* анализ текста договора;
* поиск релевантной информации через RAG;
* генерация ответа пользователю;
* подготовка юридического текста;
* обработка ошибок при некорректных файлах;
* backend-логика для взаимодействия с пользователем.
