# NormTasks — Backend Testing Notes

Repository: https://github.com/Tomaso1420/NormTasks

## Описание
Backend-проект Kanban-доски для управления проектами и задачами.

## Что можно тестировать
- регистрацию пользователей;
- авторизацию;
- JWT-токены;
- создание проектов;
- создание досок;
- создание колонок;
- создание задач;
- перемещение задач между колонками;
- роли пользователей;
- ошибки доступа;
- работу с базой данных.

## Моя роль
Разработчик и тестировщик учебного проекта.

## Стек
Python, FastAPI, PostgreSQL, Docker, JWT, REST API.
