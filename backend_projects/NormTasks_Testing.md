# NormTasks — Kanban Board

Repository: https://github.com/Tomaso1420/NormTasks

## Описание проекта

NormTasks — backend-проект Kanban-доски для управления проектами и задачами.
Проект реализован как REST API с авторизацией пользователей, работой с проектами, досками, колонками и задачами.

## Что реализовано

* регистрация и авторизация пользователей;
* JWT-авторизация;
* работа с проектами;
* работа с досками;
* создание и изменение колонок;
* создание и изменение задач;
* перемещение задач между колонками;
* работа с ролями пользователей;
* взаимодействие с базой данных PostgreSQL;
* запуск проекта через Docker.
