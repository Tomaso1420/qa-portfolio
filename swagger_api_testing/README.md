# Swagger API Testing

## Описание
Проект по ручному API-тестированию через Swagger.

## Что сделано
- проанализирована Swagger-документация API;
- составлен чек-лист проверок;
- выполнены позитивные и негативные проверки;
- оформлены баг-репорты;
- указаны шаги воспроизведения, фактический и ожидаемый результат.

## Файлы
- `checklist.xlsx` — чек-лист проверок API.
- `bug_report.xlsx` — баг-репорты по найденным дефектам.

## Инструменты
Swagger, REST API, Excel, чек-листы, баг-репорты.
