## Проекты

### 1. Postman API Testing — Restful Booker

Папка: `01_postman_restful_booker`

Практический проект по тестированию REST API сервиса Restful Booker через Postman.

Что сделано:

* создана Postman-коллекция;
* реализованы запросы для работы с бронированиями;
* настроено получение списка бронирований;
* настроено получение бронирования по id;
* реализовано создание токена авторизации;
* добавлены запросы на создание, обновление и удаление бронирования;
* использованы переменные окружения;
* реализовано сохранение `booking_id`;
* реализовано сохранение `token`;
* добавлена Basic-авторизация;
* добавлены базовые проверки статус-кодов;
* добавлена валидация JSON-схемы через tv4.

Инструменты:

`Postman`, `REST API`, `JSON`, `Basic Auth`, `Token Auth`, `tv4`.

### 3. Java Autotests — Student API

Папка: `03_java_autotests_student_api`

Практический проект по реализации и автоматизации проверки требования для API студентов.

Требование:

при поиске студента через `GET /api/v1/students/{id}`, если студент отсутствует в базе данных, API должен возвращать статус `404`.

Что сделано:

* проанализировано требование;
* реализована обработка ситуации, когда студент отсутствует в базе данных;
* добавлено исключение для отсутствующего студента;
* подготовлены базовые автотесты;
* автоматизирована проверка негативного сценария;
* проверяется поведение сервиса при поиске студента по несуществующему id.

Инструменты:

`Java`, `Spring Boot`, `JUnit 5`, `Mockito`, `AssertJ`, `Maven`, `Git`.

#### DocValidator Bot

Repository: https://github.com/Tomaso1420/doc-validator-bot

ИИ-бот для анализа юридических документов и подготовки юридических текстов. Бот имел реальных юзеров, в ходе разработки были приобретены навыки работы по Scrum

Реализовано:

* загрузка пользовательских документов;
* ИИ-Анализ документа на риски;
* обработка документов разных форматов;
* поиск релевантной информации через RAG;
* генерация ответа пользователю;
* обработка некорректных файлов;
* Составление шаблонов юр.документов
* backend-логика взаимодействия с пользователем.

Стек:

`Python`, `FastAPI`, `RAG`, `ChromaDB`, `Telegram Bot API`, `OCR`.
