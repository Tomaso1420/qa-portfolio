# Java Autotests: Student API

## Описание
Учебный проект по реализации и автоматизации проверки нового требования для API студентов.

## Требование
При поиске студента через `GET /api/v1/students/{id}`, если студент отсутствует в базе данных, API должен возвращать статус `404`.

## Что сделано
- реализована обработка ситуации, когда студент отсутствует в базе данных;
- добавлено исключение для отсутствующего студента;
- подготовлены автотесты;
- проверяется негативный сценарий поиска студента по несуществующему id.

## Инструменты
Java, Spring Boot, JUnit 5, Mockito, AssertJ, Maven, Git.

## Файл
- `studenttestexamples.zip` — архив с учебным Java-проектом и автотестами.
