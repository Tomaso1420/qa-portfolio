# Postman API Testing

## Описание
В этой папке находятся учебные Postman-коллекции, выполненные в рамках курса по QA.

## Состав
- `Booker.postman_collection.json` — коллекция для тестирования Restful Booker API.

## Restful Booker
В коллекции Booker реализованы:
- получение списка бронирований;
- получение бронирования по id;
- создание токена авторизации;
- создание бронирования;
- обновление бронирования;
- удаление бронирования;
- работа с переменными окружения;
- Basic Auth;
- token-based authorization;
- базовые проверки статус-кодов;
- проверка JSON-схемы через tv4.

## Инструменты
Postman, REST API, JSON, Basic Auth, Token Auth, tv4.
